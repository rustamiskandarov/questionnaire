# Questionnare

The idea of this project was to build app like stackoverflow

## Features

- JWT auth
- Roles, Permissions
- Telegram bot

## Installation

```bash
$ npm install
```

## Running the app

```bash
# start docker containers
$ docker-compose up

# migrations for default data
$ npm run db: migrate

# add default data
$ npm run db: seed

# development
$ npm run start

# watch mode
$ npm run start:dev
```
